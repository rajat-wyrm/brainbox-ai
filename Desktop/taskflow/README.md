# TaskFlow – Task Management Dashboard

TaskFlow is a lightweight productivity dashboard built using **HTML, CSS, and JavaScript**.  
It allows users to manage daily tasks efficiently, track progress, and maintain focus throughout the day.

This project demonstrates core web development concepts such as **DOM manipulation, responsive layout, client-side storage using LocalStorage, and dynamic UI rendering**.

---

## Project Features

### Task Management
- Add new tasks
- Delete tasks
- Edit existing tasks
- Mark tasks as completed

### Task Organization
- Assign categories to tasks (Work, Study, Personal, Fitness)
- Set priority levels (Low, Medium, High)
- Add optional due dates

### Productivity Dashboard
- Total task counter
- Completed tasks counter
- Pending tasks counter
- Completion percentage

### Smart Features
- Task search functionality
- Category filtering
- Priority filtering
- Dynamic progress bar

### Data Persistence
All tasks are stored using **LocalStorage**, so tasks remain saved even after refreshing the page.
